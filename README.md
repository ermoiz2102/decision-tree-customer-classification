# 🌳 Decision Tree Customer Classification

## 📌 Overview

This project uses a Decision Tree Classifier to predict whether a customer will default based on financial and demographic features. The model is visualized to interpret decision-making rules.

---

## 🎯 Objective

* Build a classification model using Decision Tree
* Understand feature-based decision rules
* Evaluate model performance

---

## 📂 Dataset

The dataset contains customer-related features such as:

* Initial Payment
* Last Payment
* Credit Score
* House Number

Target:

* Defaulter (Yes/No)

---

## 🔍 Steps Performed

* Data Loading
* Feature Selection
* Train-Test Split
* Model Training
* Tree Visualization
* Model Evaluation

---

## 📊 Results

* Decision Tree achieved an accuracy of **93.94%** on test data
* Optimal performance was achieved using `max_depth = 5`
* Tree visualization provided clear and interpretable decision rules
* Model shows good balance between accuracy and overfitting
